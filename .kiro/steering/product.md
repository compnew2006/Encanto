# Product Overview: Encanto

## Purpose
Encanto is a professional-grade, team-based messaging operations platform built for WhatsApp automation, team collaboration, and operational analytics. It enables organizations to manage live customer conversations at scale through an intelligent inbox, assignment engine, and audit trail.

## Target Users
- Customer support teams using WhatsApp Business
- Operations managers needing visibility and control over agents
- Admins managing multi-instance WhatsApp deployments

## Key Features
- Real-time multi-agent inbox with chat assignment and status tracking
- WhatsApp instance management (QR pairing, connection health)
- Role-based access control with fine-grained permission scopes
- Campaign broadcast engine with per-recipient outcome tracking
- Agent analytics and audit logging
- License enforcement with cleanup mode for quota overages

## Business Objectives
- Replace fragmented WhatsApp tools with a single operations hub
- Provide compliance-grade audit trails for every sensitive action
- Support multi-tenant organizations with isolated data and scoped visibility

## Success Metrics
- Zero missed messages in the inbox under concurrent agent load
- Campaign delivery rate > 98%
- Full audit trail coverage for all admin actions
